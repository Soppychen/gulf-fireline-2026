# 《海湾火线 2026》音频资产清单

版本：0.2  
用途：Demo 第一关“闪电攻击”临时原创音频包  
授权状态：项目内程序合成原创素材，可作为占位或 Demo 素材继续迭代。

## 1. 音乐

| 文件 | 用途 | 建议播放方式 |
| --- | --- | --- |
| `assets/music/music_menu_loop.ogg` | 主菜单循环音乐，120 BPM，较克制的电子脉冲 | 玩家首次点击后循环播放 |
| `assets/music/music_stage01_loop.ogg` | 第一关普通段循环音乐，132 BPM，电子军鼓街机风格 | 关卡开始后循环播放 |
| `assets/music/music_boss_loop.ogg` | Boss 战循环音乐，140 BPM，增强低频和警报纹理 | Boss 入场后切换循环 |
| `assets/music/music_victory_stinger.ogg` | 胜利结算短音乐 | 胜利结算进入时播放一次 |
| `assets/music/music_failure_stinger.ogg` | 失败短音乐 | 失败结算进入时播放一次 |

同名 `.mp3` 与 `.wav` 文件一并提供，浏览器优先使用 `.ogg`，兼容需要时回退 `.mp3`。

## 3. 混音建议

- 音乐默认音量建议 `0.35-0.45`。
- 玩家机炮默认音量建议 `0.18-0.25`，自动射击密集时进一步降低。
- 导弹锁定默认音量建议 `0.45-0.6`，但每个威胁窗口只播放一次或设置冷却。
- 玩家导弹发射音量建议 `0.25-0.35`，敌方 SAM 发射音量建议 `0.35-0.45`。
- 如果暂时只保留一个 `sfx_missile_launch` 占位，也应在代码中为玩家导弹和敌方 SAM 设置不同音量和音高。
- 爆炸同屏多次触发时建议做总线限幅，或同类爆炸同时最多播放 2-3 个实例。
- Web 端音乐与循环音效应在玩家点击“开始游戏”或首次触摸后播放，符合浏览器自动播放限制。
