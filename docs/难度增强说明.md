# 《海湾火线 2026》难度增强说明

版本：0.1  
范围：Demo 第一关“闪电攻击”  
状态：已实装为当前 Demo 默认难度增强版

## 1. 调整目标

试玩反馈显示：玩家即使不按方向键，也能较安全地通过较长流程，说明敌机武器威胁不足，关卡没有持续迫使玩家移动。

本轮调整目标：

- 让玩家“停在原地”不再安全。
- 提高敌机武器存在感。
- 增加地面防空和导弹压迫。
- 保持仍可通过，不把主线第一关直接改成极限惩罚关。

## 2. 已实装改动

### 2.1 敌人射击逻辑

文件：`src/entities/Enemy.ts`

已将敌方普通射击从“固定向下射击”改为“朝玩家当前位置瞄准射击”。

不同敌人新增差异：

- 巡逻无人机：单发瞄准弹。
- 拦截无人机：双发小角度夹击弹。
- 轻型战机：三发散射瞄准弹。
- 防空炮：五发扇形压制弹。

设计意图：

- 玩家停在底部中央时，会被瞄准弹覆盖。
- 玩家需要横向移动和低速精密移动，而不是只靠火力清屏。
- 地面防空炮成为区域压制单位，而不只是背景目标。

### 2.2 敌人数值

文件：`src/data/enemies.ts`

已提高部分敌人速度、开火频率和耐久。

调整方向：

- 巡逻无人机速度提高，开火更频繁。
- 拦截无人机速度提高，并新增开火能力。
- 轻型战机开火间隔缩短，生命略增。
- 防空炮开火间隔缩短，生命略增。
- 导弹车和 SAM 发射架发射频率提高。

### 2.3 敌方导弹

文件：`src/entities/Missile.ts`

已增强敌方地对空导弹：

- 初段加速更快。
- 追踪速度提高。
- 转向能力增强。

仍保留：

- 发射前预警。
- 有限燃料时间。
- 可通过变向、拉扯和 EMP 躲避。

### 2.4 关卡波次

文件：`src/data/levels.ts`

已在第一关中后段加入额外压力波次：

- 31.8 秒：中心无人机压制。
- 47.2 秒：右侧追加无人机群。
- 64.2 秒：中心拦截无人机突入。
- 94 秒：追加 SAM 发射架。
- 129.5 秒：追加导弹车。

设计意图：

- 主线关卡中段不再有过长安全窗口。
- 玩家获得导弹补给后，有更多高威胁地面目标可打。
- Boss 前维持一定紧张感。

## 3. 当前难度定位

当前版本建议定义为：

> 普通 Demo 的强化版，目标是“玩家需要认真移动才能过关”。

它不是地狱难度，也不应作为最终唯一难度。后续建议在菜单中提供：

- 普通：给新玩家体验流程。
- 老兵：当前增强版或略高。
- 地狱：独立短时挑战模式。

## 4. 后续调参建议

如果仍然偏简单：

- 继续缩短轻型战机和防空炮开火间隔。
- 增加敌弹速度 10%-15%。
- 降低导弹补给掉落数量。
- 减少护盾和维修出现次数。
- 让 Boss 阶段 2 开始加入 SAM 锁定。

如果偏难：

- 降低防空炮五发扇形为三发。
- 增加导弹锁定预警时间。
- 降低拦截无人机速度。
- 在 70 秒左右增加一次维修包。

## 5. 验收标准

增强版第一关应满足：

- 玩家完全不动，通常会在 30-60 秒内被击中或死亡。
- 普通玩家第一次试玩可到达中段，但不一定通关。
- 熟悉操作后，应能通过走位、拾取和技能完成通关。
- 死亡原因应可理解，不能主要来自无预警瞬杀。
