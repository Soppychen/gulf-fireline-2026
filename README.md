# 海湾火线 2026

Web 首发的纵向卷轴空战射击 Demo。当前版本实现第一关“闪电攻击”的可玩原型，使用 TypeScript、Phaser 3 和 Vite。

## 运行

```bash
npm install
npm run dev
```

打开：

```text
http://127.0.0.1:5173/
```

构建发布包：

```bash
npm run build
```

## 当前 Demo 内容

- 主菜单、游戏场景、结算场景。
- 键盘移动：WASD 或方向键。
- 触摸移动：拖拽飞机跟随。
- 自动射击、火力升级、维修、护盾、倍率拾取物。
- Space 或右下按钮触发短时护盾并清除敌方弹体。
- P/Esc 或右上按钮暂停。
- 火力道具可把机炮从 Lv1 升到 Lv4，HUD 会显示当前武器等级。
- 导弹补给道具会增加 `MSL` 弹药，玩家机载导弹会自动锁定 Boss、SAM、导弹车等高威胁目标。
- 巡逻无人机、拦截无人机、轻型战机、防空炮、导弹车、雷达站。
- 固定式 SAM 发射架和导弹车会先显示锁定预警，再发射敌方地对空追踪导弹。
- 移动指挥车队 Boss 约 105 秒入场，含阶段切换、扇形弹幕、导弹和无人机召唤。
- 分数、连击、受击次数、击毁率、评级和最高分 localStorage。
- 接入 `assets/` 内的原创临时美术、背景、特效、音效和音乐资源。

## 内容边界

当前实现使用架空表达和原创临时资源，不包含真实人物、真实坐标、真实旗帜、军徽或现实可操作战术信息。
